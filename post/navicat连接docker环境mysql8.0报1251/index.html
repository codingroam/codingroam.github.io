(设置远程登录密码123456,密码别设置的太简单，否则知道你服务器ip就随便登录你的mysql了，不然就给服务器3306端口加上来源限制，真的血的教训)

// 设置远程连接权限
grant all on *.* to 'root'@'%';

//刷新权限
flush privileges;
